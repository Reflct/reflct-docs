# 3DGS Simplified
 
Welcome to Reflct, a simplified platform for managing and publishing 3DGS scenes on the web. Reflct is currently in beta and is still under development. [Please join our discord server](https://discord.gg/rfYNxSw3yx) for updates and help. 

If you would like to request an invite to the beta, email willie@reflct.app.
